PartialProtocol

@dynamicMemberLookup
public protocol PartialProtocol

The protocol that Partial and PartialBuilder implement. It is not provided for consumers of Partial to implement this protocol; it is provided to ensure the API of Partial and PartialBuilder are identical, and to provide default implementations.
