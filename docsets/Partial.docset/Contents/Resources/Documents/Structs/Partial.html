  • set(value:for:)

    Uses updateValue(_:forKey:) to ensure the value is set to nil, rather than removed from the dictionary, which would happen if the subscript were used. This ensures that the backingValue‘s value will not be used when a backingValue is set and a key is explicitly set to nil, and also allows the value(for:) function to get a value of nil from the dictionary.
